# Digital Tools for Learning (DTL)

This repository contains a collection of **Digital Tools for Learning (DTL)** — interactive, browser-based resources designed to support teaching and learning.

## 📁 Repository Structure

Each folder contains an HTML file with the content and logic for one DTL. Some folders may also include associated assets (images, scripts, etc.).

## 🌐 Viewing the Tools Online

All tools are published via **GitHub Pages** and can be accessed directly in your browser — no installation required.

Use the following URL format to open any tool:
```
https://[username].github.io/[repo-name]/[folder-name]/[filename].html
```

**Example:**
```
https://nidiana.github.io/LabsDiscussion/Venturi_Effect/Venturi.html
```

## 💻 Running Locally (Offline Use)

You can also download and run any tool on your local computer without an internet connection:

1. Download the folder of the tool you want to use.
2. Open the `.html` file in your browser.
3. **Important:** Update the file paths for any images referenced in the HTML code so they point to the correct local locations.

Once the paths are updated, the tool will run fully offline.

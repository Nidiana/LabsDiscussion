Question 1: True Value Assessment

Which device gives the true value for the flow rate of the pump?

A. Orifice plate
B. Rotameter
C. Volume displacement method
D. None of them

✓ Correct Answer: D None of them

Every measurement is an approximation of the true value - if the instrument and measurements are very good, then it will give a very close approximation.

Question 2: Systematic Error Identification

Which of the following is most likely a systematic error with the volume displacement method?

💡 Key Concept:

Systematic errors are consistent and repeatable — think bias.

A. Variations in stopwatch timing by different students
B. Misreading the volume scale by 0.05 L each time because parallax error
C. Air bubbles forming during the experiment
D. Small splashes when pouring water into the tank

✓ Correct Answer: B

Parallax error (consistently misreading the scale) introduces the same bias every time. This makes it systematic. Random timing differences or splashes are random errors, not systematic.

Question 4: Calibration

What could be classed as a calibration error in the manometer & orifice plate?

A. Random fluctuations in the flow rate
B. A person misreading the display
C. Digital manometer not zeroed

✓ Correct Answers: C

If the digital manometer is not zeroed before measurement, all readings are shifted, causing a systematic calibration error. Using wrong dimensions would also introduce systematic error, but random fluctuations and misreading are not calibration errors.

Question 5: Error Propagation

Looking at your summary table with relative uncertainties (in the Data Processing Tool), which method increases the processed relative uncertainty compared with the raw data relative uncertainty?

A. Orifice plate
B. Rotameter
C. Volume displacement
D. None of them
E. I can't say

Answer Information:

An orifice plate calculation, which relies on a square root function (Q∝ √ (Δh)), will inherently reduce the relative uncertainty to half of the input measurement's uncertainty. In contrast, a rotameter provides a direct reading, so the measurement uncertainty carries over directly to the final result without change. Finally, the volume displacement method (Q=V/t) compounds the uncertainties from two separate variables (volume and time), resulting in a final uncertainty that is necessarily larger than that of either individual measurement. The propagation formula is:

δQ/Q = √[(δV/V)² + (δt/t)²]
